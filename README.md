# Portfólio de Estudos em Cibersegurança

Este repositório reúne minha documentação prática de estudos em cibersegurança, com foco em aprendizado real, desenvolvimento de raciocínio técnico e construção de portfólio para entrada no mercado.

## Objetivos

- Consolidar conhecimentos técnicos de forma prática
- Documentar laboratórios, salas e exercícios resolvidos
- Desenvolver raciocínio analítico em segurança ofensiva e defensiva
- Construir um portfólio técnico público no GitHub
- Acompanhar minha evolução na área de cibersegurança

## Estrutura do Repositório

- `tryhackme/` → documentações de salas e laboratórios do TryHackMe
- futuramente: outras trilhas, labs, anotações e projetos práticos

## Metodologia de Documentação

Cada sala é documentada individualmente, com foco em:

- objetivo do laboratório
- conceitos aprendidos
- ferramentas utilizadas
- técnicas praticadas
- principais aprendizados

> Este repositório não publica respostas, flags ou soluções sensíveis de plataformas de treinamento.  
> O foco está no aprendizado, no raciocínio técnico e na evolução prática.
